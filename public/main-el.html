Γεγονότα

Αυτό που κάνει το GDevelop μοναδικό και τόσο εύκολο στη χρήση, είναι τα γεγονότα. Τα γεγονότα είναι ένας ισχυρός τρόπος για να εκφραστεί η λογική του παιχνιδιού σου, χωρίς να χρειάζεται να μάθεις κάποια γλώσσα προγραμματισμού.

Όταν το πλήκτρο Space πιεστή, παίζουν οι κινήσεις και ο ήχος του χαρακτήρα. Εάν μια βόμβα αγγίξει το χαρακτήρα, και τα δύο καταστρέφονται.

Create HTML5, mobile and native games

GDevelop flexible architecture allows you to choose if you want to create an HTML5 game that can be played on the web and exported to iOS and Android, or a native game for your operating system. Or activate both platforms and export your game everywhere!

Κάνε λήψη και εγκατέστησε το GDevelop για Windows:

Κάνε λήψη για Windows

επίσης διαθέσιμο without installer (7zip format)

Το GDevelop είναι διαθέσιμο για Ubuntu 64bits μόνο!

Άνοιξε ένα τερματικό πατώντας Ctrl + Alt + T.

Στη συνέχεια, πληκτρολόγησε (και πάτα enter μετά από κάθε γραμμή):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

Τελείωσες! Πληκτρολόγησε GDevelop στη γραμμή αναζήτης της εκκίνησης για να ξεκινήσεις.

Κάνε λήψη για OS X

This is a beta: bugs may be present! You need OS X Yosemite to run GDevelop.

Δυστυχώς, προς το παρόν το GDevelop είναι μια εφαρμογή για επιτραπέζιο υπολογιστή, αλλά εάν θέλεις να δημιουργήσεις παιχνίδια οπουδήποτε, σε έχουμε καλυμένο!

Το GDevApp είναι μια ριζικά καινοτόμα, πολύ ισχυρή εφαρμογή βασισμένη πάνω στο GDevelop. Μπορείς να ξεκινήσεις τη δημιουργία ενός παιχνιδιού σε 5 λεπτά - απευθείας από το πρόγραμμα περιήγησης, το smartphone ή το tablet!

Πήγαινε στο GDevApp.com
