Ereignisse

Was GDevelop einzigartig und so einfach zu bedienen macht, sind die Ereignisse. Ereignisse sind eine leistungsstarke Möglichkeit, die Logik Ihres Spiels auszudrücken, ohne eine Programmiersprache zu lernen.

Wenn die Leertaste gedrückt wird, werden die Charakteranimation und der Ton gespielt. Wenn eine Bombe den Charakter berührt, werden sie beide zerstört.

HTML5 & systemeigene Spiele

GDevelops flexible Architektur ermöglicht es Ihnen auszuwählen, ob Sie ein HTML5-Spiel erstellen möchten, das im Internet gespielt werden kann oder für Android oder als ein systemeigenes Spiel für Ihr Betriebssystem exportiert werden soll. Oder aktivieren Sie beide Plattformen zu, und exportieren Sie Ihr Spiel überall hin!

Herunterladen und installieren von GDevelop für Windows:

Download für Windows

Ebenfalls erhältlich Ohne Installtationsprogramm

GDevelop is available for Ubuntu 64bits only!

Öffnen Sie ein Terminal durch Drücken Ctrl + Alt + T.

Dann tippe (und drücke Enter nach jede Zeile):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

Du bist fertig! Gebe GDevelop in die Suchleiste ein, um zu starten.

GDevelop erhalten Sie unter:

Fedora, Arch, OpenSuse
