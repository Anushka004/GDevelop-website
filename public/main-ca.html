Esdeveniments

El que fa GDevelop únic i tan fàcil d'usar són els esdeveniments. Els esdeveniments són una poderosa manera d'expressar la lògica del seu joc, sense haver d'aprendre un llenguatge de programació.

Quan es prem Espai, l'animació de personatges i un so es reprodueixen. Si una bomba toca el caràcter, tots dos són destruïts.

Create HTML5, mobile and native games

GDevelop flexible architecture allows you to choose if you want to create an HTML5 game that can be played on the web and exported to iOS and Android, or a native game for your operating system. Or activate both platforms and export your game everywhere!

Descarregar i instal·lar GDevelop per a Windows:

Descarregar per a Windows

també disponible without installer (7zip format)

GDevelop està disponible per 64bits Ubuntu només!

Obriu un terminal prement Ctrl + Alt + T.

A continuació, escriviu (i premeu Intro després de cada línia):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

Vostè està! Escrigui GDevelop a la barra de cerca de llançament per començar.
