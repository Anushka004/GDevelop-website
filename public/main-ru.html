События

Что делает GDevelop уникальным и простым в использовании - так это события. События - мощное средство для выражения логики вашей игры, без необходимости изучения языка программирования.

Когда нажата клавиша "Пробел", проигрывается звук и анимация персонажа. Если бомба касается персонажа, они оба уничтожаются.

Скачать для Windows

также доступны без установки

GDevelop доступен только для Ubuntu 64bits!

Откройте терминал, нажав Ctrl + Alt + T.

Затем введите (и нажмите клавишу enter после каждой строки):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

Готово! Чтобы запустить, введите GDevelop в строке поиска приложений.
