Etkinlikler

GDevelop'u benzersiz ve çok kolay kılan etkinlikler vardır. Etkinlikler, programlama dilini öğrenmeden oyunun mantığını ifade etmede en etkili yoldur.

Boşluk tuşuna basıldığında, karakter animasyonu ve ses yürütülür. Bomba karaktere dokunursa, ikisi birlikte yok olur.

Create HTML5, mobile and native games

GDevelop flexible architecture allows you to choose if you want to create an HTML5 game that can be played on the web and exported to iOS and Android, or a native game for your operating system. Or activate both platforms and export your game everywhere!

Windows için GDevelop'u kurun:

Windows için indirin

ayrıca kullanılabilir without installer (7zip format)

GDevelop sadece64 Bit Ubuntu üzerinde kullanılabilir!

Terminali açmak için şuna basabilirsiniz Ctrl + Alt + T.

Ardından yazın (her satırdan sonra enter tuşuna basın):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

İşte! Çalıştırmak için arama çubuğuna GDevelop yazın.
