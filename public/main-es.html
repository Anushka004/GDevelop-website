Eventos

Lo que hace a GDevelop único y tan fácil de usar son los eventos. Los eventos son una forma poderosa de expresar la lógica de tu juego, sin tener que aprender un lenguaje de programación.

Cuando la barra espaciadora es presionada, la animación del personaje y la reproducción de un sonido son iniciadas. Si una bomba toca al personaje, ambos son destruidos.

Juegos HTML5 & nativos

La arquitectura flexible de GDevelop te permite elegir si quieres crear un juego HTML5 que puede ser jugado en la web o exportado para Android, o un juego nativo para tu sistema operativo. O activa ambas plataformas y exporta tu juego a todas partes!

Descargar para Windows

también disponible sin instalador

GDevelop is available for Ubuntu 64bits only!

Abre una terminal presionando Ctrl + Alt + T.

Luego escribe (y presiona Enter después de cada linea):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

¡Terminaste! Escribe GDevelop en la barra de lanzamiento/búsqueda para comenzar.
