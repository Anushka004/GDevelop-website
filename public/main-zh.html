事件

让 GDevelop 变得独特和如此易于使用的是事件。事件是一个强大的方式来表达你游戏里的逻辑，而无需学习编程语言。

当按下空白建时，角色动画和声音会被播放。如果一颗炸弹触碰到字符，它们会被摧毁。

GDevelop is available for Ubuntu 64bits only!

透过按键打开终端 Ctrl + Alt + T.

然后打入 (并在每行之后按输入键)：

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

你完成了 ！在启动搜索栏中打入 GDevelop 来开始。
