Eventos

O que faz GDevelop original e tão fácil de usar são os eventos. Eventos são uma forma poderosa para expressar a lógica do seu jogo, sem ter que aprender uma linguagem de programação.

Quando a tecla de espaços for pressionado, a animação de personagem e som são executados. Se uma bomba toca o personagem, eles são ambos destruídos.

jogos HTML5 & nativos

A arquitetura flexível do GDevelop permite que você escolha se deseja criar um jogo HTML5 que pode ser reproduzido na web e exportado para o Android, ou jogo nativo para o seu sistema operativo. Ou permitir que as duas plataformas exportem o seu jogo para todos os lugares!

Baixar e instalar o GDevelop para Windows:

Download para Windows

também disponível Sem instalador

Abra um terminal pressionando Ctrl + Alt + T.

Em seguida, digite (e pressione enter após cada linha):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

Já terminou! Digite GDevelop na barra de pesquisa de lançamento, para começar.
