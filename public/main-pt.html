Eventos

GDevelop é único e muito fácil de usar por causa dos "eventos". Eventos são um jeito incrível de traduzir a lógica que você que para o seu jogo, sem ter que falar a língua da sua máquina.

Quando você aperta espaço, ambas animação e som do personagem acontecem. Se uma bomba toca o personagem, ambos se destroem.

Create HTML5, mobile and native games

GDevelop flexible architecture allows you to choose if you want to create an HTML5 game that can be played on the web and exported to iOS and Android, or a native game for your operating system. Or activate both platforms and export your game everywhere!

Baixar e instalar o GDevelop para Windows:

Download para Windows

também disponível without installer (7zip format)

GDevelop está disponível somente para Ubuntu 64 bits

Abra o terminal pressionando Ctrl + Alt + T.

Depois digite (e pressione 'Enter' após cada linha):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

Já terminou! Digite GDevelop na barra de pesquisa de lançamento, para começar.

Você pode obter o GDevelop para:

Fedora, Arch, OpenSuse
Arch (a partir do código fonte)

Este é um pacote não oficial

Lembre-se que o GDevelop é primeiro testado no Ubuntu. A sua opinião é bem-vinda no fórum.
